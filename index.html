lights-out

Lights Out game - phaser.js html5 implementation

Game

A simple board game, where you have to make all tiles the same color. Clicking a tile toggles the tile's color and all it's for neighbours. More details about the game can be found in wikipedia

Running

  1. Download the project
  2. Open index.html (IE, Firefox. See about Chrome the note below).

If you want to open with Chrome, the files have to be served through a web server - nginx, iis, apache.
